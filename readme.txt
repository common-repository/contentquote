== Description == 

Content Quote automatically finds posts that others tweeted and searches for quotes in each post.  

<h2>Twitter Search Quote Tweet Button</h2>

We periodically search twitter for links to your blog.  If users quote your content, we log the tweets in a database.  When we find these quotes, we add a blockquote of the quote after the paragraph on your post with a tweet button underneath the blockquote.  

<h2>BlockQuote Tweet</h2>

When this feature is turned on, the plugin searches for any blockquote text on a post that is under 120 characters and adds a tweet button underneath the blockquote text.  

== Installation == 

<h2>Using The WordPress Dashboard</h2>

<ol>
	<li>Navigate to the 'Add New' Plugin Dashboard</li>
	<li>Select contentquote.zip from your computer</li>
	<li>Upload</li>
	<li>Activate the plugin on the WordPress Plugin Dashboard</li>
</ol>

<h2>Using FTP</h2>
<ol>
	<li>Extract contentquote.zip to your computer</li>
	<li>Upload the contentquote directory to your wp-content/plugins directory</li>
	<li>Activate the plugin on the WordPress Plugins dashboard</li>
</ol>
